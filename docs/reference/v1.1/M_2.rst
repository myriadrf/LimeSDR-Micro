M.2 edge connector
##################

LimeSDR Micro M.2 2280 v1.1 board communicates with the host system via M.2 edge connector. 
It is configured to use PCIE-based SSD drive pinout when sloted into key M+B socket, 
but is also compatible with key M and key B sockets. 

LimeSDR Micro M.2 2280 v1.0 connector pinout and signals according to the specification is given in Table 14.
