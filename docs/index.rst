LimeSDR Micro is small form factor PCIe Software-Defined Radio (SDR) board, available in two different form factors:

* `LimeSDR Micro M.2 2280 <M.2>`_ 
* `LimeSDR Micro Mini PCIe <mPCIe>`_ 
 
Please see the above links for detailed hardware documentation.

For software support see the `Beta Programme <beta>`_ documentation.
